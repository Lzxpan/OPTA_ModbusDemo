# 貢獻與協作規範（繁體中文）

為避免語言不一致，請遵守以下規範：

## 1) 語言統一
- Issue、PR 標題、PR 內文、Code Review 回覆、Commit 說明：**一律使用繁體中文**。
- 除非文件本身是第三方原文引用，否則請勿使用英文段落作為主要說明。

## 2) 建議貼給 AI 代理的固定指令
每次開新對話時，建議在第一段直接貼上：

> 你所有回應、commit 訊息、PR 標題與 PR 內文，全部都必須使用繁體中文。若你發現不是繁體中文，請先自我修正後再送出。

## 3) PR 自我檢查
送出 PR 前請確認：
- [ ] PR 標題是繁體中文
- [ ] PR 內文全部是繁體中文
- [ ] 測試章節為繁體中文，並包含實際命令
- [ ] 若無法執行測試，已在繁體中文中說明環境限制

## 4) 若再次出現英文怎麼辦
- 在 Review 留言要求「請改為繁體中文後重送」。
- 以本倉庫的 PR Template 為準，未符合語言檢查視為未完成。
